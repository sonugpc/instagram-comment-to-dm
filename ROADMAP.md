# Roadmap

This roadmap is optimized for three goals:

1. Launch a reliable hosted SaaS.
2. Make the public repo worth sharing.
3. Differentiate from broad chatbot platforms by becoming an Instagram campaign operating system.

## Phase 0: Foundation

Status: merged through issue [#5](https://github.com/im-anishraj/instagram-comment-to-dm/issues/5).

- Email magic-link auth.
- Workspace tenancy.
- Instagram account connection.
- Keyword campaigns.
- Meta webhook processing.
- BullMQ worker.
- Stripe subscriptions.
- Vercel cron.
- Landing page.
- Production deployment docs.

## Phase 1: Launch Readiness

Issue: [#7 Production readiness](https://github.com/im-anishraj/instagram-comment-to-dm/issues/7)

- Resolve dependency audit issues or document safe exceptions.
- Make monthly usage and rate limits atomic under concurrency.
- Add worker health checks and operational alerts.
- Add admin diagnostics for webhook, billing, token, and worker failures.
- Add Privacy Policy, Terms, Data Deletion, and Meta App Review support pages.

## Phase 2: Campaign OS

Issue: [#8 Campaign OS repositioning](https://github.com/im-anishraj/instagram-comment-to-dm/issues/8)

- Reposition automations as campaigns.
- Add campaign goals and outcomes.
- Improve onboarding around first campaign creation.
- Make dashboards more agency/client friendly.

## Phase 3: Viral Template Loop

Issue: [#9 Public campaign templates](https://github.com/im-anishraj/instagram-comment-to-dm/issues/9)

- Add public template library.
- Add seeded templates for high-intent niches.
- Route "Use this template" into signup.
- Let users contribute templates.
